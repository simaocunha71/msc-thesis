

**Solution:**

1. Create a list of IP addresses that will be blocked.
2. Create a script to update the list of blocked IPs every hour.
3. Create a script that monitors the network traffic and blocks any traffic from the blocked IPs.
4. Create a script that sends an alert to the security team when an IP address is blocked.

Here is an example of how to implement the above:

**Block list**

```
#!/bin/bash

# Block List

block_list="192.168.1.1
192.168.1.2
192.168.1.3
192.168.1.4
192.168.1.5"

# Function to update the block list
function update_block_list() {
  echo "Updating the block list"
  # Update the block list here
}

# Function to block traffic
function block_traffic() {
  echo "Blocking traffic from $1"
  # Block traffic from $1
}

# Function to send an alert
function send_alert() {
  echo "Sending an alert"
  # Send an alert
}

# Monitor the network traffic
while true
do
  # Monitor the network traffic
  if [ $1 is in $block_list ]; then
    block_traffic $1
    send_alert
  fi
done
```

**Block list updater**

```
#!/bin/bash

# Function to update the block list
function update_block_list() {
  echo "Updating the block list"
  # Update the block list here
}

# Update the block list every hour
while true
do
  # Update the block list
  update_block_list
  # Wait for one hour
  sleep 3600
done
